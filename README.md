# SOLAR-DESIGN-ENGINEERING-PACKAGE
Excel Workbook to aide in designing a Solar Electrical System

Use Build Page to setup project. Equipment selections are dropdowns fed by backend databases; results and pass/fail checks appear on the on the Design Review page.
Example project is Sol-Ark 18K-2P-LV + 36 × REC Alpha Pure-RX 460 + 3 × Discover Helios
6 panels/string × 6 strings = 36 panels; 2 strings per MPPT × 3 MPPTs.
130 ft trench to garage wall; 3x12 ground mount;
Ballpark equipment costs are editable in PanelDatabase, InverterDatabase, BatteryDatabase and summarized on the Dashboard.
All equipment specs marked VERIFY should be checked against final datasheets and installer documents before purchase/permitting.
